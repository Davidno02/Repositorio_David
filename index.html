DEFINICIÓN

Es un teclado de conceptos que permite configurar el teclado mediante plantillas para adaptarse a las necesidades de cada usuario.

IntelliKeys USB se conecta a un puerto USB del ordenador y proporciona acceso a cualquier persona con discapacidad física, visual o cognitiva que tenga dificultades para utilizar un teclado estándar

CARACTERÍSTICAS

  • Compatible con Macintosh y Windows.
  • Trabaja simultáneamente con el teclado estándar.
  • Funciona con cualquier software de acceso al teclado.
  • Opciones de personalización con la configuración de plantillas o con el panel de control.
  • Las actualizaciones se pueden descargar desde la Web.
  • Interfaz con pulsador adecuada y flexible.
  • Se conecta y desconecta sin interrupción.
  • Se activa o desactiva mientras está conectado.
  • Estuche especial.
  • Protectores de teclado de plástico ayuda a los estudiantes a seleccionar las teclas correctas.
